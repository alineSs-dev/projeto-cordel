# Projeto Cordel Moderno

Página desenvolvida com HTML e CSS como parte das atividades práticas do curso.

O projeto apresenta o poema Cordel Moderno em uma página web com efeito parallax, explorando recursos de estilização e posicionamento de elementos utilizando apenas HTML e CSS.

## Tecnologias

- HTML5
- CSS3

## Funcionalidades

- Estrutura semântica
- Efeito parallax nas imagens de fundo
- Uso de fontes externas
- Layout responsivo
